# ML-DL-Portfolio
Проекты по машинному и глубинному обучению, которые были сделаны на 1-2 курсе и по работе в рекламе, недавно начал собирать их в единый пул на гитхабе

### 1) CNN для классификации MNIST

**Задача**  
Классифицировать рукописные цифры из датасета MNIST

**Ход решения**  
- CNN с 3 свёрточными слоями + BatchNorm + Dropout  
- Использован оптимизатор SGD с Nesterov momentum  
- Обучение на GPU 
- Рассчитаны метрики Accuracy, Precision, Recall, F1

**Результат**  
Accuracy > 99% Сравнение Adam и SGD показало, что SGD с Nesterov momentum 0.95 чуть лучше умеет в обобщение в данном кейсе


### 2) Two Tower Recommendation System (MovieLens)

**Задача**  
Построить рекомендательную систему на основе архитектуры Two Tower (DSSM) для предсказания, понравится ли пользователю фильм, и генерации персонализированных рекомендаций

**Ход решения**  
- Загружены данные MovieLens 100k (943 пользователя, 1682 фильма, 100k оценок)  
- Подготовлены различные признаки (например)  
  - Для пользователей: возрастные группы, пол, профессия, жанровые предпочтения, активность
  - Для фильмов: количество жанров, год выпуска, средний рейтинг, популярность, взвешенный рейтинг 
- Реализована Two Tower архитектура с отдельными башнями для пользователей и товаров, нормализацией эмбеддингов  
- Позитивные примеры: рейтинг = 5, негативные: рейтинг ≤ 2 (с балансировкой классов)  
- Обучение на GPU с оптимизатором AdamW и scheduler

**Результат**  

AUC = 0.9, Accuracy = 0.82

Модель неплохо фильмы, которые пользователь с высокой вероятностью оценит положительно

### 3) Прогнозирование стоимости квартир (CatBoost)

**Задача**  
Построить модель для предсказания рыночной стоимости квартир на основе текстовых объявлений о продаже

**Ход решения**  
- Извлечено 50+ признаков из «грязных» текстовых полей (метро, площадь, описание, лифт, адрес)  
- Добавлена логика: если этажность > 5 и лифт не указан — лифт добавляется и тп  
- Целевая переменная прологарифмирована для нормализации распределения  
- Обучен CatBoost с подбором гиперпараметров

**Результат**  
MAE = 5.2 млн. руб, что неплохо на датасете с разбросом от 1 млн. до 2.5 млрд., главное, что ошибка на квартирах среднего ценового сегмента сильно меньше за счет отсутсвия выбросов

### 4) Предсказание года выпуска музыки (MLP)

**Задача**  
По аудио-признакам трека предсказать год его выпуска (регрессия)

**Ход решения**  
- Загружен датасет YearPredictionMSD (90 признаков, ~500k записей)  
- Признаки отмасштабированы  
- Построена глубокая MLP с BatchNorm и Dropout  
- Обучение на GPU с динамическим уменьшением learning rate

**Результат**  
Test RMSE = 9, что является хорошим результатом для такого объёма данных


### 5) RNN для предсказания следующих символов

**Задача**  
Обучить RNN предсказывать n следующих символов в тексте

**Ход решения**  
- Текст: «the quick brown fox jumps over the lazy dog»  
- Символы закодированы one-hot  
- Реализована RNN с формулами:  
  `h = tanh(Wxh·x + Whh·h_prev + bh)`,  
  `y = Why·h + by`  
- Реализован backprop через градиенты  
- Обучение 300 эпох с softmax loss

**Результат**  
Loss снизился с 163 до 0.06 Модель успешно генерирует осмысленные последовательности символов

### 7) Работа с признаками и линейная регрессия (Salary + Insurance)

**Задача**  
Построить модели линейной регрессии для двух задач:
- Предсказание зарплаты по опыту работы (датасет Salary)
- Предсказание страховых выплат по медицинским параметрам (датасет Insurance)

**Ход решения**  
- Проведён EDA: гистограммы, точечные графики, анализ корреляций
- Для зарплат применён MinMaxScaler
- Для страховок выполнено кодирование категориальных признаков (sex, smoker, region)
- Обучены линейные регрессии, оценены метрики MAE, RMSE, R²
- Проведён отбор признаков для Insurance: добавлены age_group, bmi_category, age×bmi, штраф за курение

**Результат**  

**Salary:**  
- MAE Test = 0.047, R2 Test = 0.98 (на отмасштабированных данных)

**Insurance:**  
- Базовые признаки: MAE = 4423, R2 = 0.74  
- После отбора признаков: MAE = 3033, R2 = 0.82

Качество модели Insurance улучшено на 31% за счёт генерации новых признаков
