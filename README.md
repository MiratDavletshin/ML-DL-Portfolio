# ML-DL-Portfolio
Проекты по машинному и глубинному обучению. В основном учебные проекты и немного рабочих по рекламе

## 1. CNN для классификации MNIST

**Задача**  
Классифицировать рукописные цифры из датасета MNIST.

**Ход решения**  
- CNN с 3 свёрточными слоями + BatchNorm + Dropout.  
- Использован оптимизатор SGD с Nesterov momentum.  
- Обучение на GPU (10 эпох).  
- Рассчитаны метрики Accuracy, Precision, Recall, F1.

**Результат**  
Accuracy > 99%. Сравнение Adam и SGD показало, что SGD с Nesterov momentum 0.95 чуть лучше умеет в обобщение в данном кейсе

## 2. Прогнозирование стоимости квартир (CatBoost)

**Задача**  
Построить модель для предсказания рыночной стоимости квартир на основе текстовых объявлений о продаже.

**Ход решения**  
- Извлечено 50+ признаков из «грязных» текстовых полей (метро, площадь, описание, лифт, адрес).  
- Добавлена логика: если этажность > 5 и лифт не указан — лифт добавляется и т.п.  
- Целевая переменная прологарифмирована для нормализации распределения.  
- Обучен CatBoost с подбором гиперпараметров.

**Результат**  
Относительная ошибка ~2.5%. Модель предсказывает стоимость квартиры с высокой точностью.

## 3. Предсказание года выпуска музыки (MLP)

**Задача**  
По аудио-признакам трека предсказать год его выпуска (регрессия).

**Ход решения**  
- Загружен датасет YearPredictionMSD (90 признаков, ~500k записей).  
- Признаки отмасштабированы.  
- Построена глубокая MLP с BatchNorm и Dropout.  
- Обучение на GPU с динамическим уменьшением learning rate.

**Результат**  
Test RMSE ≈ 9.0 лет. Модель ошибается в среднем на 9 лет, что является хорошим результатом для такого объёма данных.


## 4. RNN для предсказания следующего символа

**Задача**  
Обучить RNN предсказывать следующий символ в тексте на уровне символов.

**Ход решения**  
- Текст: «the quick brown fox jumps over the lazy dog».  
- Символы закодированы one-hot.  
- Реализована RNN с формулами:  
  `h = tanh(Wxh·x + Whh·h_prev + bh)`,  
  `y = Why·h + by`.  
- Реализован backprop через градиенты.  
- Обучение 300 эпох с softmax loss.

**Результат**  
Loss снизился с ~163 до ~0.06. Модель успешно генерирует осмысленные последовательности символов.

## 5. Классификатор площадок РСЯ (сравнение моделей)

**Задача**  
Классифицировать площадки РСЯ по 6 признакам (бинарная классификация).

**Ход решения**  
- Данные: 1650 строк, дисбаланс классов 64%/36%.  
- Признаки отмасштабированы.  
- Обучены три модели:  
  - MLP (3 слоя, BatchNorm, Dropout)  
  - CatBoost  
  - XGBoost  
- Сравнение по Accuracy и F1-score.

**Результат**  

| Модель    | Accuracy | F1-score |
|-----------|----------|----------|
| NN        | 0.7970   | 0.8241   |
| CatBoost  | 0.9758   | 0.9812   |
| XGBoost   | 0.9697   | 0.9765   |

Лучший результат показал **CatBoost**. Деревья превзошли нейросеть по F1
